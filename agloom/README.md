# agloom (Python package)

This directory is the **`agloom`** import package: LangGraph-oriented agents, the AGP protocol (`agloom.protocol`), runtime bridge, observability hooks, and execution patterns.

- **Install:** `pip install agloom`
- **Docs:** [agloom.readthedocs.io](https://agloom.readthedocs.io)
- **Repo:** [github.com/HELLOMEDHIRA/agloom](https://github.com/HELLOMEDHIRA/agloom)

The PyPI wheel ships library modules only (tests and examples under this tree are excluded from the distribution per `pyproject.toml`). Browse examples under `agloom/examples/` in the repository.
