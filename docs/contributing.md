# Contributing

See the [Contributing Guide](https://github.com/HELLOMEDHIRA/agloom/blob/main/CONTRIBUTING.md) for:

- Development setup (Python 3.12.x, uv)
- Running tests (`uv run pytest tests`)
- Linting and formatting (`ruff` on `agloom`, `agloom_cli`, and `tests`)
- Type checking (`pyrefly`)
- Commit conventions (Conventional Commits)
- Pull request process
