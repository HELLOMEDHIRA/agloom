# agloom-web

Vite + React workspace UI for AGP-native sessions: chat, observability, runtime visualization.

## Development

From **`agloom_web/`** (this directory is the npm package root):

```bash
npm install
npm run build
npm run dev
```

AGP wire types live in **`src/lib/agp/types.ts`** — keep it identical to `agloom_cli/src/types/agp.ts`.

- **Repo:** [github.com/HELLOMEDHIRA/agloom](https://github.com/HELLOMEDHIRA/agloom) · package path `agloom_web/`
